# How to Install This Chrome Extension

This extension is currently running in "Developer Mode" and is not yet published on the official Chrome Web Store. Follow these simple steps to install it locally on your browser:

1. **Open the Extensions Page**
   Open Google Chrome, type the following URL into your address bar, and press Enter:
   `chrome://extensions/`

2. **Enable Developer Mode**
   In the top-right corner of the Extensions page, you will see a toggle switch labeled **Developer mode**. Click it to turn it **ON**. 
   *(This will reveal a few new buttons at the top of the page).*

3. **Load the Extension**
   Click the **Load unpacked** button that appeared in the top-left corner.

4. **Select the Folder**
   A file browser window will open. Navigate to the folder where you saved/cloned this project:
   
   Select that folder (make sure you pick the folder that contains the `manifest.json` file inside it) and click **Select Folder**.

5. **Pin the Extension (Optional but Recommended)**
